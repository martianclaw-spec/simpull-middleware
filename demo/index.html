API
SimPull Middleware API — This demo consumes the middleware's stable REST API.

GET /api/cars — All installed cars with normalized names and preview URLs
GET /api/tracks — All installed tracks/layouts with preview URLs
GET /api/car-preview/:carId — Car preview image (never 404s)
GET /api/track-preview/:track — Track preview image
GET /api/track-preview/:track/:layout — Layout preview image
GET /api/status — Service health + scan stats
POST /api/rescan — Re-scan the AC directory
GET /api/config — Current config (acPath, port)
PUT /api/config — Update config

This demo is a minimal consumer only. The API is designed to be consumed by kiosks, overlays, launchers, and third-party sim platforms independently.
